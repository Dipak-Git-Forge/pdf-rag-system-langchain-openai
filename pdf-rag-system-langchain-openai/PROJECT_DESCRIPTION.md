# Project Description

## Short GitHub Description
A Retrieval-Augmented Generation (RAG) system for PDF documents using LangChain, FAISS, Hugging Face embeddings, and OpenAI.

## Resume / LinkedIn Description
Developed a Retrieval-Augmented Generation (RAG) system for PDF-based knowledge retrieval using LangChain, FAISS, Hugging Face embeddings, and OpenAI. The system extracts text from PDF documents, chunks and embeds the content into a vector database, retrieves semantically relevant context through similarity search, and generates grounded responses using a large language model.

## One-Line Interview Summary
I built an end-to-end PDF question-answering system where the model does not answer purely from memory, but first retrieves relevant content from the document and then generates a context-aware response.
