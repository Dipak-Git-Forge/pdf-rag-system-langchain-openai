# LinkedIn / GitHub Post Draft

Built a **PDF-based RAG system** using **LangChain, FAISS, Hugging Face embeddings, and OpenAI**.

### What the project does:
- extracts text from a PDF
- splits the text into chunks
- converts chunks into embeddings
- stores them in a FAISS vector database
- retrieves the most relevant context for a query
- generates a grounded answer using an LLM

### Tech stack:
Python | LangChain | FAISS | Sentence Transformers | OpenAI | Streamlit

### Example use case:
Ask a question like:
> "a bread that uses wheat flour and is suitable for a dinner party"

The system retrieves the most relevant recipe context from the PDF and generates an answer based on that context.

This project helped me better understand:
- Retrieval-Augmented Generation (RAG)
- vector databases
- semantic search
- grounding LLM outputs with external documents
